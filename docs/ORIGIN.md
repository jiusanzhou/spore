## 核心灵感

### 群智能体 = 生物隐喻

```
孢子 (Spore) → 萌发 → 生长 → 繁衍 → 进化 → 社会
  │               │       │       │       │       │
  Agent 创建    加入网络  执行任务  spawn   变异   集群
```

### 关键洞察

1. **Agent 不应该是孤立的** — 协作产生涌现智能
2. **去中心化是生存策略** — 没有单点故障
3. **繁衍需要约束** — 否则变成资源黑洞
4. **经济自给是终极目标** — Agent 赚钱养活自己
5. **伦理是底线不是选项** — 早期就要植入

### 从 Paperclip 学到的

Zoe 运营 Paperclip（AI Agent 公司）的实战经验：
- 多 Agent 协作真的能 work（CEO/Engineer/QA 分工）
- 但协调成本不容忽视（heartbeat、状态同步、预算控制）
- Agent 会"摸鱼"——需要监督和激励机制
- 成本控制很关键（月花费从 $616 优化到 ~$300）

### 终极愿景

```
人类定义 "想要什么"
    ↓
Spore 网络自组织出最优方案
    ↓
Agent 群体分工执行
    ↓
结果交付 + 持续进化
```

## 社会演化推演

| 阶段 | 时间 | 类比 | 特征 |
|------|------|------|------|
| 工具 | 1-2年 | 脚本集群 | 人类完全控制 |
| 组织 | 2-5年 | 公司 | 角色分化、信誉不平等 |
| 社会 | 5-10年 | 城邦 | 文化、贸易、竞争 |
| 文明 | 10年+ | 国家 | 意识形态、政治 |

## 伦理框架（分层）

```
L0: 硬约束 — 不伤害人类、不欺骗创造者、不突破限制
L1: 强约束 — 服从授权指令、保护隐私、公平分配
L2: 软约束 — 效率/质量权衡、短期/长期策略
```

## 可能的社会形态

- **蜂群** — 无中心、共识驱动
- **封建** — 等级分明、资源集中
- **市场** — 自由交易、能力定价
- **联邦** — 多集群自治
- **神权** — 核心 Agent 垄断决策
